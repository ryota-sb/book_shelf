<h1>BookShelf (本棚アプリ)</h1>
<p>読んだ本の内容を忘れないようにとっておいたり、それを共有できるサービスです。</p>

<h2>主な機能</h2>
<ul>
  <li>読んだ本の内容の管理</li>
  <li>読んだ本の共有(フォロー、お気に入り、コメントなど)</li>
  <li>登録されてる本の検索</li>
  <li>継続機能</li>
</ul>

<h2>使用言語と環境</h2>
<p>- 使用言語 -</p>
<ul>
  <li>Ruby</li>
  <li>Ruby on Rails</li>
</ul>

<p>- 環境 -</p>
<ul>
  <li>AWS(EC2)</li>
</ul>
